# C_Learning

Implement some basic functions in c for learning purpose:
1. Linked list functions
2. String functions
3. File operations
etc

This uses scons build system.

Install and build instructions:
1. virtualenv -p python3 .venv3;
2. source .venv3/bin/activate;
3. pip install scons;
4. scons;
5. ./test_exe
